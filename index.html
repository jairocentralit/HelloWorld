Login





Este código cria uma página de login com campos para nome de usuário e senha. Quando o botão "Entrar" é clicado, a função checkPassword() é chamada. Se a senha inserida for "rita@123", um alerta de login bem-sucedido é exibido, caso contrário, um alerta de senha incorreta é exibido. Lembre-se de que este é apenas um exemplo simples para fins de demonstração. Em um ambiente real, você deve adicionar medidas de segurança mais robustas, como criptografia de senha e autenticação segura.
